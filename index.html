Light-first editorial framework

Quiet surfaces, bold type, and the kind of spacing that lets the page breathe.

This framework is built around the Apple web language found in editorial and developer pages: soft greys, large rounded cards, restrained gradients, and a layout rhythm led by typography rather than decorative effects.

Build with the framework

Simple class names, disciplined defaults, and enough flexibility for a full editorial page.

Start with `container`, `section`, `stack`, `grid`, and `card`. Then layer in `text-gradient`, `gradient-card`, and `gradient-section` where the page needs a visible but still controlled moment of emphasis.
